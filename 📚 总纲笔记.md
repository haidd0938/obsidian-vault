> 系统里的坑、骚操作、冷知识，记在这里免得忘。

### Provider 配置
- **主力**: DeepSeek (`deepseek-chat`) ¥58.13余额(截至4/30)
- **备选**: NVIDIA NIM (免费, 40次/分钟, 2026-10-18到期)
- **视觉**: Gemini (`AIzaSy...Ya30`)
- ⛔ Provider **永不**自动切换（NVIDIA切DeepSeek出过胡言乱语bug，已删cron）
- 切换命令：手动找贾维斯

### 系统暗坑
- MacBook Pro **Intel芯片** → 无MPS/GPU加速，本地ML跑不动
- SD/FLUX等图像生成 → 优先API方案，别碰本地推理
- HTTP代理: `127.0.0.1:6478`
- TG可能走不了代理，有问题排查

### 快捷命令/端口

| 服务 | 地址 | 启动命令 |
|------|------|---------|
| Hermes Web UI | http://localhost:8648 | 已配launchd |
| n8n | http://localhost:5678 | `~/n8n/start-n8n.sh` 或 `n8n-start` |
| **OneAPI中转站** | **http://localhost:3010** | **`docker start one-api`** |
| 语音模式 | TG发语音→STT→处理→TTS回复 | 已配置Edge TTS(中文Xiaoxiao) |

- 飞牛NAS：FN ID=haidd0938, 用户=haidd, 密码=haidd
- 有Docker，可装Hermes，远程穿透不稳定(kubi)
