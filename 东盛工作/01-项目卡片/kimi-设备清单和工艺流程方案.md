## 二、生产工艺流程

基于传统手工工艺  结合工业化生产要求，建议采用以下工艺流程：

```
原料验收 → 浸泡 → 磨浆 → 过滤 → 沉淀 → 淀粉精制 → 调配搅拌 → 
蒸煮糊化 → 冷却回醒 → 成型/捏制 → 切分 → 真空包装 → 
巴氏杀菌/高温灭菌 → 冷却 → 风干 → 装箱入库
```

### 关键工艺控制点：

| 工序 | 工艺参数 | 控制要点 |
|------|----------|----------|
| **原料浸泡** | 48小时，水温15-25℃ | 荞麦充分软化，直链淀粉比例调整  |
| **磨浆** | 石磨/电动石碾细磨 | 保留传统工艺特色，浆液细腻度控制  |
| **过滤沉淀** | 多次换水沉淀 | 去除杂质，获得纯净淀粉  |
| **蒸煮糊化** | 文火慢煮，持续搅拌 | 温度85-95℃，搅拌1小时左右，色泽由白转粉红  |
| **回醒** | 加盖静置冷却 | 淀粉凝胶结构形成，口感绵软  |
| **真空包装** | 抽真空密封 | 延长保质期至2个月（配合杀菌） |
| **杀菌** | 高温灭菌锅或巴氏杀菌 | 确保微生物指标合格  |

## 四、车间布局建议

```
原料仓库 → 原料处理区（浸泡、清洗、磨浆）→

内包装区（成型、切分、真空包装）→ 
杀菌区（高温灭菌）→ 外包装区 → 成品仓库
```

**关键控制**：
- 生熟分开，避免交叉污染 
- 内包装区建议设置十万级洁净车间
- 配备更衣室、洗手消毒设施
