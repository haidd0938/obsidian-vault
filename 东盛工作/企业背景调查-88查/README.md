## 🛠️ 项目背调标准流程

当你拿到一个**甘肃项目实施单位**的名称时：

### Step 1：查基础信息（30秒）
打开 **https://88cha.com** → 搜企业名
- ✅ 确认企业存续（不是注销/吊销）
- ✅ 确认法人、注册资本、成立时间
- ✅ 确认统一社会信用代码

### Step 2：查风险记录（1分钟）
侧栏点击 **"查风险/查老赖"**
- ❌ 有没有失信被执行记录？
- ❌ 有没有限制高消费？
- ❌ 有没有行政处罚？
- ❌ 有没有司法诉讼？

### Step 3：查经营实力（1分钟）
- 侧栏 **"查招投标"** → 看最近中标了什么项目，判断实力
- 侧栏 **"查专利"** → 看有没有技术壁垒/创新力
- AI背调报告 → 看AI对企业的综合评价

### Step 4：查关联方（30秒）
- 看股东的关联企业（有没有皮包公司嫌疑）
- 看对外投资（是不是空壳还是真有项目）

### Step 5：输出评估结论

### 风险等级：🟢低 / 🟡中 / 🔴高
- 失信记录：有/无
- 诉讼风险：有/无（XXX起诉讼）

## ⚙️ 技术细节

### 88查 API 接口

> 88查是纯Web端工具，**没有开放API**，直接用浏览器操作。

如果需要自动化查询，可以用Python请求：

```python
import requests
import json

def search_company(name):
    """搜索企业基本信息"""
    url = "https://88cha.com/api/company/search"
    resp = requests.get(url, params={"keyword": name}, timeout=10)
    return resp.json()

def ai_background_check(name):
    """AI企业背调"""
    url = "https://88cha.com/api/ai/background"
    resp = requests.post(url, json={"company": name}, timeout=30)
    return resp.json()
```

> ⚠️ **注意：** 以上API路径是猜测的，实际接口需通过抓包获得。**推荐直接浏览器操作**，免费且稳定。

## 🚫 避坑提示

- **不要只看法人名字** — 有些皮包公司法人是挂名的
- **一定要查"查风险"** — 失信被执行是最大红线
- **分公司也要查** — 分公司有问题总公司一样背锅
- **招投标记录要看最近** — 半年内没中过标说明业务萎缩
