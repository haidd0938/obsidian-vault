# 🎬 东盛视频目录

> 东盛建筑所有视频产出和素材统一管理。

## 目录结构

| 目录 | 内容 | 说明 |
|------|------|------|
| **EPC建筑视频/** | README.md（记录笔记） | 视频生产记录 |
| **项目文档/东盛建筑视频/** | 已发布的视频成品 + 中间素材 | 脚本、场景图、字幕、配音片段、最终合成视频 |

- 项目文档/东盛建筑视频/ 里是正式产出的成品视频
- 最新视频记录在 [[EPC建筑视频/README]]
