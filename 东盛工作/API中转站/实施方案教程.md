## 二、你需要掌握的操作

### 2.1 登录管理后台

浏览器打开 http://localhost:3001 或走公网 Tunnel 地址
账号 root / 密码 oneAPI8836

### 2.2 查看渠道（Channels）

登录后点左侧「渠道」→ 能看到当前已配置的 7 个渠道
每个渠道是一个供应商（比如 SiliconFlow、DeepSeek 官方等）

要点：
- 渠道可以随时启用/禁用
- 模型映射（Model Mapping）：渠道里的模型名可以改，比如供应商那叫 deepseek-ai/DeepSeek-V4-Flash，你可以映射成 deepseek-v4-flash，客户按你起的名字调用
- 权重（Weight）：多个渠道提供同一模型时，按权重轮询

### 2.3 管理 Token

左侧「Token」→ 可以看到已生成的密钥

每个 Token：
- 可以限制额度（比如设 $10 上限）
- 可以限制可用模型（比如只让用 DeepSeek V4）
- 可以绑定到特定用户
- 可以设置过期时间

客户给你钱后，你就在这给他们开一个 Token，设好额度。

### 2.4 新增渠道（如果要加新供应商）

左侧「渠道」→ 「添加渠道」

必填：
- 名称：随便起（比如 硅基流动）
- 类型：选 OpenAI（绝大部分都是这个）
- 密钥：供应商给你的 API Key
- 代理地址（Base URL）：供应商的 API 地址
  - SiliconFlow：https://api.siliconflow.cn/v1
  - DeepSeek：https://api.deepseek.com/v1
  - OpenRouter：https://openrouter.ai/api/v1
  - Groq：https://api.groq.com/openai/v1
  - Google Gemini：https://generativelanguage.googleapis.com/v1beta/openai/
- 模型重定向（Models）：填入你能用的模型名，一行一个

### 2.5 查看日志

左侧「日志」→ 可以看到所有用户的调用记录
- 每次调用花了多少钱（绿色是成功，红色是失败）
- 调用量统计
- 这是你算利润的核心数据

## 四、你必须知道的细节（坑）

### 4.1 Tunnel 地址会变

trycloudflare.com 的免费 Tunnel 地址每次重启都会变。
如果你重启了 Docker 或电脑，隧道会分配到新地址。

应对：
- 跑 `bash ~/.hermes/scripts/start-tunnel.sh` 重新启动
- 启动后用 `docker logs cloudflared-tunnel 2>&1 | grep trycloudflare` 查看新地址
- 通知客户新地址

长期方案：绑定自己的域名到 Cloudflare（需要你有 Cloudflare 账号），这样地址固定不变。

### 4.2 One API 重启后自动恢复

docker 容器设置的是 --restart unless-stopped，
电脑重启或 Docker 重启后 One API 会自动恢复。

### 4.3 利润核算脚本要等有数据才准

刚启动没数据时，利润看板会显示零。
需要真实客户调用几次后，日志里才有数据。

### 4.4 成本控制

SiliconFlow 是最便宜的进货渠道，但也要留意用量：
- DeepSeek V4 Flash：¥1/1M tokens
- 一个客户每天消耗 500 万 tokens，成本才 ¥5

如果你的 Token 没限制额度，可能出现一个客户把整个月预算消耗完的情况。
所以给客户开 Token 时务必设置额度上限。

### 4.5 模型名称映射技巧

客户不需要记住供应商的命名。你想怎么叫就怎么叫：
- deepseek-v4-flash → 供应商 deepseek-ai/DeepSeek-V4-Flash
- gpt4o → 供应商 openai/gpt-4o
- claude35sonnet → 供应商 anthropic/claude-sonnet-3.5

在渠道编辑里的「模型重定向」设置就行。

## 六、进阶路径

### 6.1 固定域名（稳定地址）

如果你有一个自己的域名（比如 api.nixiaoban.com），
注册 Cloudflare 免费账号 → Zero Trust → Networks → Tunnels → 创建 Named Tunnel → 绑定你的域名
这样地址就是 https://api.nixiaoban.com，不会变。

### 6.2 自动对账报表

利润核算脚本支持：
```bash
# 日报
python3 ~/.hermes/scripts/profit-tracker.py --daily-report

# 月度分析
python3 ~/.hermes/scripts/profit-tracker.py --monthly-analysis
```

### 6.3 客户自助注册

One API 支持开放注册，登录面板 → 设置 → 允许注册
开启后别人自己注册，自己充值，省去你手动开 Token 的麻烦。

注意：开放注册前一定要设好默认额度，避免被刷。
