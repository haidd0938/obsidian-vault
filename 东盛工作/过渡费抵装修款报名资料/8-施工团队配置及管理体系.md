# 施工团队配置及管理体系

## 一、拟投入本项目的组织架构

```
                       项目经理
                          │
          ┌───────────────┼───────────────┐
          │               │               │
      技术负责人       施工员长       质量安全员
          │               │               │
    ┌─────┴─────┐    ┌────┴────┐         │
    │           │    │         │         │
   设计师   深化专员  各专业班组     │
                （水电/木工/油漆     质检+安检
                  /瓦工/安装）
```

## 二、核心岗位配置

| 岗位 | 配置数量 | 职责 | 持证要求 |
|------|:-------:|------|---------|
| 项目经理 | 1名 | 项目总负责、进度协调、甲方对接 | 二级及以上注册建造师 |
| 技术负责人 | 1名 | 技术方案、质量管控、图纸深化 | 工程师及以上职称 |
| 专职安全员 | 1名 | 现场安全巡查、安全交底、隐患排查 | 安全员C证 |
| 质量员 | 1名 | 工序验收、材料验收、质量标准把控 | 质量员证 |
| 设计师 | 1-2名 | 户型测量、方案设计、效果图出具 | 室内设计相关专业 |
| 施工员 | 按工程量配置 | 现场施工管理、班组调度 | 施工员证 |

## 三、施工班组配置

| 班组 | 配置人数（按100套批量估算） | 作业内容 |
|------|:-------------------------:|---------|
| 水电机电班 | 4组×2人=8人 | 全屋水电改造、强弱电布线 |
| 瓦工班 | 4组×3人=12人 | 墙地面找平、瓷砖铺贴、防水施工 |
| 木工班 | 3组×2人=6人 | 吊顶施工、背景墙制作、柜体安装 |
| 油漆班 | 3组×2人=6人 | 腻子批刮、乳胶漆喷涂、壁纸铺贴 |
| 安装班 | 2组×2人=4人 | 门窗安装、厨卫洁具、开关灯具 |
| 杂工及保洁 | 4人 | 材料搬运、垃圾清运、成品保护、保洁 |
| **合计** | **约40人** | 可根据项目体量灵活增减 |

## 四、施工管理体系

### 4.1 标准化流程

```
安置户选档确认 → 户型测量 → 方案设计 → 材料清单确认
    → 进场准备（成品保护） → 拆除/基层处理 → 水电改造（隐蔽验收）
    → 防水施工（闭水试验） → 瓦工铺贴 → 木工吊顶
    → 油漆涂料 → 安装工程 → 竣工验收 → 保洁交付
```

### 4.2 质量管控体系

| 阶段 | 管控措施 | 责任人 |
|------|---------|--------|
| 设计阶段 | 图纸会审、方案确认签字 | 技术负责人 |
| 材料进场 | 品牌核验、检测报告查验 | 质量员 |
| 隐蔽工程 | 100%验收，拍照留档 | 质量员+监理 |
| 防水 | 闭水试验48小时，楼下无渗漏 | 施工员+质量员 |
| 中期验收 | 完成60%工程量时，全面检查 | 项目经理+质量员 |
| 竣工验收 | 按验收标准逐项检查 | 项目经理+安置户 |

### 4.3 工期保障体系

- **材料预采购**：主材提前15天完成备货
- **多组平行作业**：按户型分组同时施工
- **日报制度**：每日向甲方汇报进度
- **周例会**：每周召开进度协调会

### 4.4 售后服务体系

| 阶段 | 时间节点 | 服务内容 |
|------|---------|---------|
| 交付 | 竣工当天 | 向安置户交付使用说明、质保卡 |
| 回访 | 交付后7天 | 电话回访满意度 |
| 跟踪 | 交付后1个月 | 上门检查使用情况 |
| 维保 | 质保期内 | 24小时报修热线随时响应 |
| 定期 | 质保期内每年 | 主动巡检一次 |
