# 分档装修报价方案

## 一、报价思路

基于公告要求的"质量优先、价格公允"原则，结合天水及陇东南地区装修市场行情，我公司制定三档标准化装修报价方案，对应安置户不同需求层次，确保价格透明、分档清晰、材料可控。

**核心原则：**
1. 报价包含人工、辅材、主材、管理费及合理利润
2. 主材按每档指定品牌，不增项不加价
3. 面积以套内实际装修面积计算
4. 过渡费抵款结算单价与现金结算单价一致，不因抵款模式上浮报价

## 四、报价说明

1. **面积计算方式**：按套内实际装修面积计算（非建筑面积），以现场实测为准。
2. **单套面积适用**：单套面积在60-130㎡范围内适用上述单价；小于60㎡按最低起步价计取；大于130㎡的异形户型，另行协商。
3. **个性化增项**：超出标准配置的内容（如额外定制柜体、智能家居集成、特殊造型施工等），按实际发生的工程量另行计价，报价清单中已列明各项单价。
4. **批量优惠**：同一项目批量装修套数超过50套，给予总价3%优惠；超过100套，给予总价5%优惠。
5. **价格有效期**：本报价自提交之日起6个月内有效。
6. **过渡费抵款**：上述单价为统一报价，过渡费抵款结算与现金结算执行相同单价标准，不因结算方式不同而调整价格。
