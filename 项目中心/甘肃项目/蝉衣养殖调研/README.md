## 调研背景

近期网络出现蝉衣（蝉蜕/金蝉/蝉花）人工养殖推广内容。经研判，该产业具有"低投入、林地利用、持续产出"特点，与天水杨家寺镇、陇南西和县的山区林果经济高度契合，具备核桃林、花椒林下套种套养条件。

## 调研方法

- 网络公开信息检索（百度百科、中药材市场数据、农业技术文献）
- 产业案例分析（山东平邑金蝉养殖产业链、河南许昌卵枝供应体系）
- 本地基础数据对比（杨家寺镇、西和县气候/地理/林果产业资料）
- 市场行情采集（中药材交易平台、电商平台卵枝/蝉蜕/金蝉价格）
