# Google 工作套件
**技能名：** google-workspace

## ⚡ 简介
通过 gws CLI 或 Python 操作 Gmail、Calendar、Drive、Docs、Sheets。

## 📋 使用方法
跟我说：
- "帮我查一下 Gmail"
- "在日历上安排一个会议"
- "在 Google Drive 里找XXX文件"
- "把数据写到 Google Sheets"

## 💡 案例
> **你说：** "帮我在日历上安排下周三上午的会议"
> **我：** Calendar API → 创建事件 → 发送邀请
