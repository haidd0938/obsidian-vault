## 核心功能

全流程 AI 短剧生产工作台：策划 → 编剧 → 分镜 → 出片

- **无限画布生产工作台** — 自由编排剧本/角色/分镜/素材/视频节点
- **三层 Agent 协作体系** — 决策层/执行层/监督层协同，质量审阅与修订反馈
- **持久化 Agent 记忆** — 本地 ONNX 向量检索的跨会话记忆系统
- **可编程供应商系统** — 直接在设置中心编写供应商 TypeScript 逻辑，无需改源码
- **章节事件图谱** — 自动提取原著章节事件并结构化存储
- **Skill 文件化配置** — 核心提示词外化为 Markdown Skill 文件，支持在线编辑
