## 使用场景

### 小红书/抖音内容创作（副业）
- 下载竞品热门视频参考 → GreenVideo（B站/小红书）或 tiktokio（抖音）
- 去除水印后重新剪辑 → MagicEraser
- 灵感搜集 → cobalt.tools 下X上的视频素材

### EPC工程（偶尔）
- 下载施工工艺参考视频 → GreenVideo 或 yt-dlp
- 培训素材整理 → yt-dlp 批量下载YouTube教程

### 台球厅宣传
- 下载台球赛事精彩片段做二创 → cobalt / GreenVideo
- 去水印 → MagicEraser
