# html-ppt — HTML PPT Studio

## 基本信息
- **仓库**: lewislulu/html-ppt-skill
- **⭐**: 4,690
- **许可证**: MIT
- **安装路径**: ~/.hermes/skills/html-ppt/

## 核心能力
| 模块 | 数量 | 说明 |
|------|------|------|
| 主题 | 36 | 含 xiaohongshu-white、corporate-clean、cyberpunk-neon 等 |
| 全 Deck 模板 | 15 | 含 xhs-post（9 页 3:4）、pitch-deck、tech-sharing、weekly-report |
| 单页布局 | 31 | 封面/目录/双栏/代码/图表/时间线/架构图等 |
| CSS 动画 | 27 | 入场动画（淡入、上升、打字机等） |
| Canvas 特效 | 20 | 粒子爆发、星空、知识图谱、神经网络等 |
| 演讲者模式 | ✅ | S 键弹出 4 磁吸卡片：当前页/下一页/逐字稿/计时器 |

## 用法
Agent 可以用一句话生成 HTML 幻灯片：
- "做一份 8 页的技术分享 slides，用 cyberpunk 主题"
- "把这个大纲变成 Pitch Deck"
- "做一个小红书图文，9 张，白底柔和风"

## 验证
已通过浏览器打开 `templates/full-decks-index.html` 验证，所有模板/主题/布局文件完整可用。

## 对老板的价值
1. **小红书图文** — 配套 xiaohongshu-white 主题 + xhs-post 9页 3:4 模板
2. **EPC汇报方案** — 投标/结算汇报直接出 HTML 幻灯片
3. **技术分享/演讲** — presenter-mode-reveal 带逐字稿模板
4. **鑫球汇推广** — pitch-deck / product-launch 模板可改
