# n8n 自动化工作流
**技能名：** n8n-standalone-docker-deployment

## ⚡ 简介
你的 n8n 运行在 localhost:5678，数据在 ~/n8n/data，用 ~/n8n/start-n8n.sh 或 n8n-start 命令启动。

## 📋 使用方法
跟我说：
- "启动 n8n"
- "帮我搭个自动化工作流"
- "n8n 跑不起来了帮我看看"

## 💡 案例
> **你说：** "帮我在 n8n 里搭一个定时爬取资讯的流程"
> **我：** 配 trigger → 配 HTTP request → 配通知节点
