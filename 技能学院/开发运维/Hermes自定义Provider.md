# Hermes 自定义 Provider
**技能名：** hermes-add-custom-provider

## ⚡ 简介
配置 Hermes 的自定义 provider：配置健康检查、API 兼容性测试，改 config.yaml。

## 📋 使用方法
跟我说：
- "帮我加一个新的大模型 provider"
- "配置 DeepSeek 的 API"
- "这个 API 怎么接进 Hermes"

## 💡 案例
> **你说：** "帮我加一个 moonshot 的 API 到 Hermes"
> **我：** 改 config.yaml → 配置 base_url/api_key → 测试连通性
