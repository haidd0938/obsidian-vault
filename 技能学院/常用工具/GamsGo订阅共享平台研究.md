# GamsGo 订阅共享平台研究 + API 中转站对比

## GamsGo 是什么？

GamsGo（gamsgo.com）是一个低价订阅共享平台，运营了7年。

核心模式：
- 批量购买 Netflix / YouTube Premium / Spotify / Disney+ / ChatGPT Plus 等服务的家庭方案
- 拆分成独立座位转售给个人用户
- 按天或按月收费

核心数据：
- 15 万付费会员
- 120 万注册用户
- 自称 98% 满意率
- 7x24 客服
- 自动发货

最近新增板块：软件工具转售（Claude、Cursor、CapCut、Runway 等 13 个产品）、ChatGPT 免费试用。

## GamsGo 的 API 板块

通过 API 接口发现的：

API 基址：`https://api.gamsgo2.com`

产品分类（从 API 返回）：
1. 订阅共享（流媒体/音乐/AI工具等）
2. 软件工具转售
3. ChatGPT 相关
4. 第三方商城（卖家入驻）

具体 API 路由：
- `/api/v1/en/categories` — 获取产品分类
- `/api/v1/en/products` — 获取产品列表（Cloudflare 防护，需真实浏览器访问）

## 与我们的 API 中转站对比

| 维度 | GamsGo | 我们的 API 中转站 |
|------|--------|------------------|
| 模式 | 订阅共享转售 | API 额度转售 |
| 毛利率 | 20-40%（平台抽成高） | 60-75%（直接进货） |
| 合法性 | 灰色地带（违反 ToS） | 合法（技术服务） |
| 售后 | 重（账号被封需处理） | 轻（纯技术问题） |
| 启动成本 | 需批量买账号 | 100元起充 |
| 用户粘性 | 低（随时换平台） | 高（接入后难迁移） |
| 市场规模 | 大但竞争激烈 | 蓝海且快速增长 |
| 我们的优势 | — | 直连国内渠道、中文化、TG客服 |

## 结论

GamsGo 的模式可以借鉴，但我们做 API 中转站的利润率和可持续性都更好。核心套路一样：
**低价进货 → 加价转售 → 赚差价**。

我们的版本更好：
- 进货成本透明（SiliconFlow 公开定价）
- 毛利率更高（60%+ vs 20-40%）
- 完全合法（API 技术服务，不涉及版权问题）
- 市场更大（AI 开发者快速增长）

## 如何借鉴 GamsGo 的运营策略

GamsGo 成功的几个点：
1. **自动化发货**：付款后秒到账，不用人工介入
2. **多SKU覆盖**：几乎所有主流服务都有
3. **阶梯定价**：按天/按月/按年，覆盖不同需求
4. **低门槛尝鲜**：每天几块钱就能用

对应到我们的 API 中转站：
- 自动化发货 -> One API 自动创建用户和 Token
- 多SKU覆盖 -> 接入更多模型（Qwen、Kimi、DeepSeek 等）
- 阶梯定价 -> 5个套餐层级
- 低门槛尝鲜 -> 免费 100 万 token/月
