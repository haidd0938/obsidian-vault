# 📄 PDF全能工具箱

> 自研工具，17个子命令

**路径**: `~/.local/bin/pdf-tool`

**功能清单**:
| 子命令 | 功能 |
|--------|------|
| `edit-text` | 编辑PDF文字 |
| `replace-image` | 替换图片 |
| `watermark` | 加水印 |
| `merge` | 合并PDF |
| `split` | 拆分PDF |
| `rotate` | 旋转页面 |
| `compress` | 压缩 |
| `encrypt` / `decrypt` | 加密/解密 |
| `extract-text` / `extract-images` | 提取图文 |
| `add-page` / `delete-page` | 增删页面 |
| `crop` | 裁剪 |
| `scale` | 缩放 |
| `info` | 查看信息 |

**依赖**: PyMuPDF 1.27
