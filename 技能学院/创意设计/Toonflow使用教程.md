## 一、首次启动设置

### 1.1 启动应用
- 打开 Launchpad 或 Applications，找到 **ToonFlow** 点击启动
- 首次启动可能提示"无法验证开发者"，去 **设置 → 隐私与安全性 → 仍要打开**

### 1.2 登录
- 默认账号: `admin`
- 默认密码: `admin123`
- **首次登录后立即修改密码**（右上角头像 → 修改密码）

### 1.3 配置模型供应商（最重要的一步）
点击 **设置中心**，配置三类模型：

| 模型类型 | 推荐配置 | 成本 |
|---------|---------|------|
| **语言模型** | DeepSeek（你已有API Key） | ~¥0 |
| **图片模型** | 可选便宜的（GPT Image 2 / 国产图片API） | 不足¥1/次 |
| **视频模型** | Seedance 2.0 / 算能云 / 腾讯混元 | 主要成本（¥50-120/条） |

> 💡 **省钱策略：** 先配 DeepSeek（语言模型零成本），图片和视频模型先选最便宜的方案试跑一条看看效果，再决定要不要升级

### 1.4 DeepSeek 配置
- 供应商选择: 输入你的 DeepSeek API Key（`sk-f07eb02d2f5940efaec92df5a6c319e2`）
- 模型选择: DeepSeek V4 Flash（又快又便宜）
- 点击测试连接，确认能通

## 二、核心工作流（策划 → 出片）

### 2.1 第一步：导入原著
1. 点击 **新建项目**，输入项目名称
2. 导入小说/剧本源文件（支持 txt / md / docx）
3. 点击 **章节事件提取** → 自动拆解章节和事件图谱

### 2.2 第二步：剧本生成（ScriptAgent）
1. 进入 **ScriptAgent** 模块
2. 系统自动生成:
   - 故事骨架（人物关系、主线、冲突点）
   - 改编策略（节奏控制、重点章节标记）
   - 结构化剧本（分幕、分场、台词）

### 2.3 第三步：分镜与素材（ProductionAgent）
1. 切换到 **ProductionAgent**
2. 在 **无限画布** 中组织:
   - 分镜节点（每个镜头的构图、角度、时长）
   - 角色节点（人物形象设定）
   - 场景节点（背景/环境）
   - 素材节点（参考图片/视频）
3. 对分镜图进行节点化精调

### 2.4 第四步：视频生成与导出
1. 确认所有节点就绪
2. 点击 **生成视频**
3. 等待视频模型渲染（速度取决于你选的供应商）
4. 预览后，可进行拼接、加字幕、调整节奏
5. 导出成品（MP4）

## 三、核心模块详解

### 3.1 无限画布工作台
- 自由拖动节点，不限制步骤顺序
- 支持回溯（改了前面的不影响后面的）
- 支持并行生产（同时做多个分镜）

### 3.2 三层 Agent 协作
| 层 | 角色 | 工作内容 |
|----|------|---------|
| 决策层 | 总导演 | 任务拆解、整体方向把控 |
| 执行层 | 编剧/美术/剪辑 | 具体内容生成 |
| 监督层 | 质检员 | 质量审阅、修改反馈、一致性检查 |

### 3.3 持久化记忆
- 跨会话记住你的创作偏好
- 基于本地 ONNX 向量检索
- 保证多轮创作的人物/场景一致性

### 3.4 Skill 文件化配置
- ScriptAgent 和 ProductionAgent 的提示词是外化的 Markdown 文件
- 可在设置中心在线编辑，无需改源码
- 想调优角色风格、台词风格的话改这里

## 四、变现思路（参考 MrLarus 模式）

### 4.1 短剧分发渠道
| 平台 | 适合内容 | 变现方式 |
|------|---------|---------|
| 小红书 | 1-3分钟精剪短剧 | 引流到私域/接商单 |
| 微信视频号 | 短剧切片/花絮 | 平台分成/带货 |
| B站 | 完整短剧/系列 | 创作激励/充电 |
| 抖音 | 竖屏短剧 | 平台分成/直播引流 |
| YouTube Shorts | 中英双语短剧 | 广告分成 |

### 4.2 低成本试水策略
1. 先用 DeepSeek 写剧本（零成本）
2. 用最便宜的图片模型生成角色定妆
3. 用免费/低价视频模型生成第1条短剧
4. 发到小红书+视频号看数据
5. 效果好再加大投入

### 4.3 成本控制参考
| 阶段 | 单条成本 | 频率 | 月成本 |
|-----|---------|------|-------|
| 试水期 | ¥50-130 | 每周1条 | ¥200-520 |
| 稳定期 | ¥30-80 | 每周3条 | ¥360-960 |
| 规模化 | ¥20-50（找便宜视频API） | 日更 | ¥600-1500 |

## 五、常见问题

### 5.1 启动报错
- 如果提示"已损坏，无法打开" → 终端运行: `sudo xattr -rd com.apple.quarantine /Applications/ToonFlow.app`

### 5.2 视频生成太慢
- 取决于你选的视频模型供应商
- 国内推荐: 算能云 (sophnet.com) / 腾讯混元 (3d.hunyuan.tencent.com)
- 国外推荐: Seedance / Runway

### 5.3 角色前后不一致
- 利用 Agent 持久化记忆功能
- 在设置中心的人物设定里写详细描述

### 5.4 提示词调优
- 去 `data/skills/` 目录编辑 Skill 文件
- 或在设置中心在线编辑 ScriptAgent/ProductionAgent 提示词
