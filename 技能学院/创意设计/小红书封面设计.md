# 小红书封面设计指南

> ⚡ **简介**：AI副业研究所专用封面模板设计规范，用Python PIL库自动生成专业封面图。

## 📋 使用方法

### 封面尺寸规范

| 类型 | 尺寸 | 说明 |
|------|------|------|
| 笔记封面 | 1000×1000px | 正方形，主流尺寸 |
| 头像 | 500×500px | 圆形裁剪 |
| 背景图 | 1500×500px | 顶部横幅 |

### 封面模板代码

```python
from PIL import Image, ImageDraw, ImageFont

cover = Image.new('RGB', (1000, 1000), '#0f0f23')
draw = ImageDraw.Draw(cover)

# 渐变背景
for y in range(1000):
    r = int(15 + (40/1000)*y)
    g = int(15 + (90/1000)*y)
    b = int(35 + (140/1000)*y)
    draw.line([(0, y), (1000, y)], fill=(r, g, b))

# 顶部高光线
draw.rectangle([0, 0, 1000, 8], fill='#00d4ff')

font = ImageFont.truetype('/Library/Fonts/Arial Unicode.ttf', 100)
font_s = ImageFont.truetype('/Library/Fonts/Arial Unicode.ttf', 36)
font_tag = ImageFont.truetype('/Library/Fonts/Arial Unicode.ttf', 40)

# 左上角品牌名
draw.text((40, 35), "AI副业研究所", fill='#00d4ff', font=font_s)

# 主标题
draw.text((500, 280), "主标题", fill='#ffffff', font=font, anchor='mm')

# 封面文件位置：/Users/mac/xiaohongshu-mcp/data/default_cover.png
```

### 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 品牌蓝 | #00d4ff | 标题/高亮 |
| 主背景 | #0f0f23 | 深色底 |
| 白色 | #ffffff | 主文字 |
| 黄色 | #ffd93d | 重点数据 |
| 红色 | #ff6b6b | 对比/标注 |
| 灰色 | #666688 | 辅助文字 |
