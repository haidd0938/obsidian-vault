- 用 `post_comment_to_feed` 回复评论者

## 💡 成交话术SOP

### 私域首单脚本（微信沟通）

```
1️⃣ 确认问题
"你现在最想用AI解决什么？获客、效率还是成本？"

2️⃣ 量化损失
"这个问题每周大概浪费你多少时间或多少钱？"

3️⃣ 展示案例
"我之前帮XX做过类似的，效果是XXX"

4️⃣ 给三档方案
- 低价试点（99元）：先试一个小功能
- 标准交付（499元）：完整解决方案
- 深度定制（1999元）：全流程+持续优化

5️⃣ 设置下一步
"今天确认需求，明天给方案，付款后进群"
```

### 交付后跟进SOP

| 时间 | 动作 |
|------|------|
| 交付时 | 附带使用说明+操作视频 |
| 3天后 | 问是否达成目标，主动修复问题 |
| 7天后 | 发额外洞察或优化建议 |
| 14天后 | 问效果如何，能否做案例 |
| 30天后 | 给续费/升级/转介绍方案 |
