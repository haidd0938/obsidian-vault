# 🔥 热点追踪工具箱

> ⚡ **简介**：三个网站解决 80% 的信息焦虑——每天 5 分钟看完全网热点，适合做自媒体、追热门、抢前排评论、信息套利。
>
> 来源：@noahduck283 推荐 + @OpenCils 补充

## 📋 三个工具一览

| 工具 | 一句话说明 | 最佳场景 |
|:---:|-----------|---------|
| 🥇 **NewsNow** | **多平台聚合**，一个页面看完知乎/微博/B站/虎扑/V2EX | 快速扫全平台热搜，找话题灵感 |
| 🥇 **今日热榜** | **类目最全**，还能看 GitHub/Product Hunt/Hacker News | 技术圈 & 产品圈热点追踪 |
| 🥇 **SoPilot** | **X(Twitter)爆帖专用**，只看爆款推文 | 抢前排评论、捕捉舆论动向 |

---

## 🥇 1、NewsNow — 多平台聚合

**网址**：[newsnow.busiyi.world](https://newsnow.busiyi.world)

### 简介
一个页面聚合 **知乎、微博、B站、虎扑、V2EX、差评、少数派** 等主流平台的最新热门内容。不用来回切 App，刷这一个就够了。

### 特点
- **全平台**：覆盖国内主流内容平台
- **实时更新**：各平台最新热门即时抓取
- **精简视图**：标题+热度，一眼扫完

### 适合场景
- 上班前刷 2 分钟，知道今天全网在聊什么
- 找选题：看看哪个话题有讨论热度
- 内容创作参考：知道大家都在说什么，才好写差异化内容

---

## 🥇 2、今日热榜 — 类目最全

**网址**：[tophub.today](https://tophub.today)  
**技术专区**：[tophub.today/c/tech](https://tophub.today/c/tech)

### 简介
**目前最全的热榜聚合**，除了常规平台（知乎/微信/微博/百度/抖音），还能看：
- **GitHub Trending** — 今天什么开源项目最火
- **Product Hunt** — 海外新产品发布
- **Hacker News** — 极客圈热帖
- **各大科技媒体** — 36氪、虎嗅、钛媒体

### 特点
- **极全**：200+ 榜单来源
- **分类**：科技/娱乐/综合/社区等分门别类
- **追踪功能**：可以追踪某事件的热度变化趋势

### 适合场景
- **搞技术 + 信息套利**：看 GitHub Trending + Product Hunt → 写工具评测（信息套利第9招）
- **研究行业趋势**：追踪 AI/科技领域的热点走向
- **找素材**：哪个话题在上升期，适合切入做内容

## 🥇 3、SoPilot — X 爆帖专用

**网址**：[sopilot.net/zh/hot-tweets](https://sopilot.net/zh/hot-tweets)

### 简介
只看 **X(Twitter) 上的爆款推文**。精选实时热门推文，不用自己刷时间线，直接看最火的那几条。

### 特点
- **只看爆款**：过滤掉普通推文，直击热门
- **实时更新**：X 上刚火的推文即时收录
- **抢前排**：适合想做 X 营销的人抢前排评论

### 适合场景
- **抢前排评论**：大 V 爆帖刚出，抢前排评论蹭流量
- **捕捉舆论动向**：看看今天 X 上在吵什么
- **找海外热点**：配合信息套利方法论，海外火什么你马上知道

## 💡 使用场景案例

### 场景1：每天早上扫热点

> **你**：贾维斯，今天有什么热点？

> **我**：自动帮你扫三个工具，汇总今日全网热门：
> - 微博热搜 TOP3 + B站热门视频 + GitHub 今日趋势项目
> - 附上每个热点的切入角度建议
> - 如果适合做短视频/评测/帖子，直接标出来

### 场景2：找选题做内容

> **你**：贾维斯，帮我看看今天 AI 圈有什么值得写的

> **我**：用今日热榜(Tech) + SoPilot 扫一圈 →
> - GitHub 今日有个新项目爆了 → 可以写评测
> - X 上某个大佬发了新观点 → 可以写解读
> - Product Hunt 今天有个新产品 → 可以写首发体验

### 场景3：信息套利全流程

> **你**：用信息套利第9招（AI工具评测），配合热点追踪工具箱

> **我**：
> 1. 今日热榜(Tech) → 看到 GitHub 有个 AI 工具今天最火
> 2. SoPilot → X 上大家都在讨论
> 3. 研究这个工具 → 写原创评测 + 联盟链接
> 4. 发到知乎/掘金/公众号 → 蹭今日热度

### 场景4：抢前排蹭流量（X/Twitter）

> **你**：贾维斯，SoPilot 上今天有什么爆帖值得互动

> **我**：扫 SoPilot →
> - 🥇 某大 V 发了新观点，评论区还在增长中
> - 🥈 某争议话题在吵，很适合插一脚带节奏
> - 给你写好回复文案，你直接去发
