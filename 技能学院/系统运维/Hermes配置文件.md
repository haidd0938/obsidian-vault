# Hermes 配置管理
**技能名：** hermes-agent

## ⚡ 简介
配置/安装/启用/禁用/修改 Hermes Agent 本身——CLI、config、providers、tools、skills、voice、gateway。你想调 Hermes 的任何设置，先加载这个技能再操作。config.yaml 是核心配置，有干净备份。

## 📋 使用方法
跟我说：
- "帮我改一下 Hermes 的配置"
- "加个新 provider"
- "关掉自动切换"
- "配置语音模式"
- "升级 Hermes"

## 💡 案例
> **你说：** "帮我把 DeepSeek 设为默认 provider"
> **我：** 加载 hermes-agent 技能 → 改 config.yaml → 重启
