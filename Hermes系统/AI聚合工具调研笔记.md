# AI聚合工具调研笔记
> 2026-04-30 · 多模型 + 笔记一体化方案

## 需求：多个大模型和笔记汇总到一起，省去切换和整理麻烦

---

## 一、聚合工具 - 让你高效切换和管理AI模型

### 全功能桌面客户端
| 工具 | 特点 |
|------|------|
| **Cherry Studio** | 支持**同时调用多个模型对比输出** |
| **Lobe Chat** | 可一键部署私有聊天界面 |
| **Noi** | 多窗口管理，体验好 |

### 浏览器插件
| 工具 | 特点 |
|------|------|
| **ChatbotsPlace** | 侧边栏集成主流模型 |
| **Multi LLM** | **一个界面同时和多个AI聊天** |
| **BrainyAI** | 侧重辅助阅读和网页摘要 |

## 二、集成笔记功能 - 记录和AI思考无缝衔接

### AI原生笔记应用
| 工具 | 特点 |
|------|------|
| **NoteGen** | 跨平台Markdown笔记，内置多模型AI助手（ChatGPT/Gemini等），自动整理碎片化信息、生成结构化笔记 |

### 笔记增强插件 / 外部整合
| 工具 | 特点 |
|------|------|
| **TopicKeeper** | 和AI对话时随手记录关键想法 |
| **Joplin Plugin: NoteLLM** | Joplin用户的AI问答和笔记总结 |

## 选择建议（对我老板的适配）

| 场景 | 推荐 | 理由 |
|------|------|------|
| **务实/追求效率** | **Cherry Studio** 或 **Lobe Chat** | 桌面客户端，功能全，适合工作流 |
| **浏览器重度用户** | ChatbotsPlace / BrainyAI | 侧边栏即开即用 |
| **知识管理需求强** | **NoteGen**（AI原生笔记）配合 **TopicKeeper** | 笔记+AI一体化 |
| **技术人员/想自建** | **OpenRouter** + 自托管 **Lobe Chat/Thunderbolt** | 完全控制权 |

> 📌 **老板偏好判断**：务实、成本敏感、偏好图形界面。如果选工具，优先桌面客户端（Cherry Studio或Lobe Chat），配合已有Hermes系统使用最合适。
