# AIRI - AI虚拟伴侣

## ⚡ 简介
开源 AI 虚拟伴侣项目，GitHub 3.8 万 ⭐。对标油管 88 万粉的 AI 主播 Neuro-sama。
自托管、支持30+模型兼容，可本地跑。

## 📋 关键信息
| 项目 | 详情 |
|---|---|
| 官网（在线试用） | https://airi.moeru.ai |
| GitHub | https://github.com/ParthJadhav/AIRI |
| 安装方式 | macOS darwin-arm64.dmg 拖拽安装 / Web版 / Docker / 自编译 |
| 核心能力 | 实时语音对话、Minecraft/Factorio等游戏陪玩、Live2D/VRM虚拟形象、记忆系统(DuckDB) |
| 模型兼容 | 30+ API — GPT/Claude/DeepSeek/Ollama/Gemini 等，OpenAI兼容格式 |
| TTS | ElevenLTS等（非免费） |
| 中文支持 | ✅ 有中文文档 |
| 系统要求 | 中低配即可，macOS/Windows/Linux 全平台 |
| 老板已有配置 | DeepSeek API + Edge TTS(free) + Ollama(gemma4/qwen2.5) = 可直接用 |

## 💡 使用场景
- **尝鲜**：浏览器打开 airi.moeru.ai 在线试用，零安装
- **桌面二次元助手**：跑个Live2D虚拟形象在桌面陪聊天/打游戏
- **语音研究**：可作为语音对话的参考实现，研究其 RAG 和记忆架构

## ⚠️ 注意事项
- 老板的 Hermes 系统已覆盖大部分核心能力（语音对话、TG集成、记忆）
- AIRI 主打虚拟形象+游戏陪玩，与老板的 EPC/副业工作流交集不大
- ElevenLabs TTS 要付费，老板现有的 Edge TTS 免费方案更合适
- 了解即可，不建议投入时间部署，除非有特定需求（如桌面虚拟形象）
