# 甘肃省投资项目备案抓取

## 概述

本技能用于从 **甘肃省投资项目在线审批监管平台**（https://tzxm.fzgg.gansu.gov.cn/）抓取备案项目信息，并根据公司资质自动匹配相关项目。

## 实时状态（2026年5月实测）

| 接口/功能 | 状态 | 说明 |
|:---------|:----|:-----|
| 平台首页 `tzxm.fzgg.gansu.gov.cn` | ⚠️ 首页200，后台500 | 首页Vue SPA可访问(显示2024年系统升级公告)，所有API返回HTTP 500 |
| 公开API `queryProjectPublicList` | ❌ HTTP 500 | 已失效(2026年4月曾返回演示数据) |
| 公开API `queryCountByRegion` | ❌ HTTP 500 | 已失效 |
| 后台API `queryPublicList` | ❌ HTTP 500 | 需Cookie写入但返回500 |
| 后台API `search` | ❌ HTTP 500 | 需Cookie写入但返回500 |
| 公示页面 `/gsgg/index.html` | ❌ 服务器无响应 | 返回"服务器暂时无响应，请稍后再试" |
| 国家平台 `new.tzxm.gov.cn` 公示页 | ✅ 可用 | 仅显示全国最新~10条记录，无省份筛选 |

**当前结论（2026-05-08更新）：** 甘肃省级平台首页HTTP 200可访问（Vue.js SPA，含登录），但所有API接口（含公开API和需要Cookie的后台API）均返回HTTP 500。JS资源文件也返回服务器无响应。公开API `queryProjectPublicList` 此前（2026年4月）曾返回过15条演示数据（更新至2025年12月），但现已完全失效。此状态自2024年9月系统升级公告后持续至今，建议关注公告中留下的客服电话。

如需站点恢复后使用：
1. 确认站点可正常访问
2. 用自有账号登录，通过F12获取Cookie
3. 保存到 `references/cookies.txt`
4. 搭配后台API使用

## 使用场景

- 每天早上定时抓取最新备案项目
- 按公司乙级资质精确匹配可承接项目
- 按优先地区（天水、陇南等）筛选排序
- 输出汇总报告供业务跟进

- 免费用户每日有查看条数限制，建议集中抓取优先地区
- 部分项目已有招标结果（成交公告），标注清楚避免无效跟进

## 脚本说明

### `scripts/crawl_projects.py` — 核心抓取脚本（⚠️ 平台下线）

原功能（待平台恢复）：
- 通过平台API列表获取备案项目（需有效Cookie）
- Cookie认证模式：读取 `references/cookies.txt`
- 按公司乙级资质精确匹配可承接项目
- 按优先地区排序
- 自动保存报告到 `~/甘肃投资项目/YYYY-MM-DD/` 目录

参数：
| 参数 | 说明 | 默认值 |
|:----|:-----|:-------|
| `--date` | 查询日期 (YYYY-MM-DD) | 今天 |
| `--keywords` | 资质关键词，逗号分隔 | 详见company_config.ini |
| `--regions` | 优先地区，逗号分隔 | 天水,陇南,定西 |
| `--pages` | 抓取页数 | 3 |
| `--output` | 输出格式 (text/json/both) | text |

### `references/company_config.ini` — 公司资质配置

配置项：
- `name` — 公司名称
- `address` — 注册地址
- `keywords` — 资质关键词（用于匹配项目）
- `preferred_regions` — 优先地区
- `output_dir` — 输出目录（默认 ~/甘肃投资项目/）

### `references/company-qualifications.md` — 公司资质详情

完整的资质证书信息、人员配置、设备清单、经营范围，用于精确匹配可承接项目类型。

> **⚠️ 2026-05-06 更新：** 此文件中的城乡规划编制乙级已被用户确认不存在，统一社会信用代码被用户指出有误，请从风鸟小程序或资质证书原件中确认最新信息后更新。

### `references/company-governance.md` — 公司经营范围和优先级规则

**核心规则：不按"资质等级限制"筛选，按实际经营范围匹配。** 甘肃东盛的业务范围远超单纯的乙级设计资质——可承接房屋建筑、市政、公路交通、水利水电、消防、智能化、勘察测量、检测鉴定、监理、全过程咨询、施工等。筛选时以经营范围关键词匹配为主。

## 匹配逻辑（按实际经营范围匹配，非按资质等级限制）

### ⚠️ 重要：匹配原则

**不要按"乙级资质能接什么"来筛选！** 用户"甘肃东盛建筑设计有限公司"的经营范围远超资质等级限制，实际可承接的业务包括：房屋建筑工程设计、市政工程设计、公路交通工程设计、水利水电工程设计、消防设施工程设计、建筑智能化系统工程设计、城乡规划、旅游规划、土地规划编制、风景园林规划设计、施工图审查、岩土工程勘察、工程测量、地形测绘、不动产测绘、地质灾害评估、建筑工程检测、市政工程检测、结构检测、节能检测、消防检测、房屋安全性鉴定、工程监理（房建/市政/公路/人防）、全过程造价咨询、可研报告编制、项目管理、项目代建、施工总承包/专业承包、特种工程施工、房地产开发图文设计等。

**筛选的正确原则：**
1. **所有建筑/土木/基础设施类项目都可能是商机**，不要因为项目投资额大就排除
2. 重点标注与**经营范围直接匹配**的项目（如房建、市政、检测等关键词）
3. 标注项目所在地是否在**优先地区**（天水、陇南等）
4. 不要使用"乙级→投资额≤1500万"这类错误逻辑来排除项目

### 优先级/匹配度评分

| 等级 | 条件 | 说明 |
|------|------|------|
| 🥇 **核心匹配** | 项目名称含 "建筑/房建/市政/检测/改造" 等核心经营关键词 | 公司最常接的业务类型 |
| 🥈 **拓展匹配** | 含 "公路/水利/园林/规划/勘察/测量/监理" | 经营范围覆盖，有资质或能力承接 |
| 🥉 **可延伸** | 项目含 "代建/全过程/咨询/可研/施工" | 作为EPC/全过程模式下的增值服务 |
| 🌟 **优先地区** | 项目建设地在天水、陇南、定西 | 加分项，重点标注 |

**不按投资额排除项目！** 东盛可承接大量大型项目的子项设计、检测、监理、咨询等，金额不是判断标准。

## 数据输出

抓取的数据保存在 `~/甘肃投资项目/` 目录下，按日期组织：
- `~/甘肃投资项目/YYYY-MM-DD/项目列表-YYYY-MM-DD.md` — 当天完整项目列表
- `~/甘肃投资项目/YYYY-MM-DD/高价值项目-YYYY-MM-DD.md` — 根据资质筛选出的高价值项目

## Cron/定时执行工作流

本技能通过 Hermes cron 定时任务执行。任务入口应优先使用浏览器数据源流程而非运行旧脚本。

### 执行流程

```
1. 收到 cron 启动信号
2. 检查 HAS_OUTPUT 标志
   ├─ HAS_OUTPUT=true  → 已有今日产出 → 回复"SILENT"（静默结束）
   └─ HAS_OUTPUT=false → 继续执行
3. 用 browser_navigate 访问剑鱼标讯数据源：
   - https://gansu.jianyu360.cn/tianshui  (天水 ⭐)
   - https://gansu.jianyu360.cn/longnan   (陇南 ⭐)
   - https://gansu.jianyu360.cn/dingxi    (定西 ⭐)
   - https://gansu.jianyu360.cn/          (全站)
4. 用 snapshot 提取页面中的项目列表
5. 按经营范围关键词筛选匹配项目
6. 保存报告到 ~/甘肃投资项目/{日期}/ 目录
7. 回复中给出摘要
```

### 报告格式

文件结构：
- `~/甘肃投资项目/{日期}/项目列表-{日期}.md` — 完整项目列表
- `~/甘肃投资项目/{日期}/高价值项目-{日期}.md` — 高价值项目筛选
