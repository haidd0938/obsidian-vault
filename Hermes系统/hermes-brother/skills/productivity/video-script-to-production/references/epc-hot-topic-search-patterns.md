# EPC Hot Topic Search Patterns

> 从 2026-05-08 和 2026-05-09 两次 cron 运行中总结的新闻检索经验。
> 目标：每天找到1-3个建筑/EPC相关热点话题，无需 API Key、无第三方依赖。

## 可用新闻源（已验证）

| 来源 | URL | 成功率 | 特点 |
|------|-----|--------|------|
| **中国新闻网 (chinanews.com.cn)** | `https://www.chinanews.com.cn/cj/gd.shtml` (财经滚动) | ✅ 高 | 无验证码，大量中文财经新闻 |
| **界面新闻 (jiemian.com)** | `https://www.jiemian.com/` | ✅ 高 | 无验证码，行业分类清晰 |
| **华尔街见闻 (wallstreetcn.com)** | `https://wallstreetcn.com/news/global` | ✅ 高 | 无验证码，高质量财经（英文为主） |
| **东方财富 (eastmoney.com)** | `https://finance.eastmoney.com/a/czqyw.html` | ✅ 中 | 无验证码但广告较多 |
| **MOHURD (住建部)** | `https://www.mohurd.gov.cn/` | ✅ 中 | 纯政策原文，curl可抓 |
| **Baidu 新闻** | `https://news.baidu.com/` | ❌ | JS渲染 + 验证码 |
| **Google 搜索** | `https://www.google.com/` | ❌ | CAPTCHA 拦截 |
| **Bing 搜索** | `https://www.bing.com/` | ❌ | 国内版重定向，国际版无结果 |

## 浏览器检索流程

```python
# 使用 browser_navigate 工具而非 curl（curl 对 JS 渲染页面效果差）

# Phase 1: 扫描主流中文新闻源
sources = [
    "https://www.chinanews.com.cn/cj/gd.shtml",  # 财经滚动 → 找"建筑/模块化/基建/城市更新"相关内容
    "https://www.jiemian.com/",  # 整体新闻 → 重点关注"地产/工业/能源"分类
    "https://wallstreetcn.com/news/global",  # 关注基础设施/能源/科技相关内容
    "https://finance.eastmoney.com/a/czqyw.html",  # 证券聚焦 → 找"基建/工程"赛道消息
]

# Phase 2: 识别EPC相关关键词
# 在中新网/界面新闻的snapshot中搜索（grep或逐项扫描）：
keywords = ["建筑", "工程", "基建", "EPC", "施工", "基础设施", "项目",
            "城市更新", "盾构", "模块化", "投标", "招标", "总承包",
            "住建部", "绿色建筑", "装配式"]

# Phase 3: 深入阅读
# 点击感兴趣的文章标题 → 获取全文内容
```

## 建筑/EPC 热点分类体系

按选题价值从高到低排列：

### 🥇 一级热点（最佳选题）
- **政策发布**：住建部/财政部/发改委联合发文 → 财政补贴、资质改革、行业新规
- **重大工程**：世界首创技术、超大型项目开工/竣工
- **行业趋势数据**：季度/年度报告（如"EPC模式采用率"、"联合体中标占比"）

### 🥈 二级热点（可用）
- **地方实践**：城市更新计划、大型基建项目
- **技术突破**：新型施工装备、建筑工业化进展
- **市场动态**：原材料价格波动、投融资风向

### 🥉 三级热点（备选）
- **企业动态**：挂靠/分包案例、资质改革下的中小企业生存
- **国际工程**：一带一路项目进展

## 选题原则

1. **EPC相关性优先** → 能自然带出"EPC总承包"概念
2. **信息差价值** → 从业者不知道的新政策/新数据
3. **争议性/话题性** → 能引发评论区讨论（如"你的工地违规分包了吗"）
4. **品牌合规** → 不碰敏感词（安全事故除非官方已定性通报）

## 标题公式

```
[重磅/突破/数据/政策/趋势] + [核心信息点] + [追问/悬念]
```

示例：
- "世界首创！三元混合气盾构装备问世，中国基建EPC再添'大国重器'"
- "中央财政2026年城市更新：每城最高12亿！不足15城入选，谁是赢家？"
- "住建部今日出手！EPC违规分包被严查，2026资质改革下谁被洗牌？"
