# TikTok/抖音 → 可执行内容模式库

> 合并自 `media/tiktok-patterns-to-skill` 技能。用于将抖音/TikTok爆款模式分析并套用到老板的赛道（建筑EPC、鑫球汇台球）。

## 已收录套路库

### 套路一：「AI宠物」——结果展示+吸引想赚钱的人
**原始模式：** AI生成宠物内容 → 涨粉 → 卖课/带货
**核心：** 省钱+解压+低门槛

**台球版：** 日记体「辞职开台球厅第X天」，展示真实收入
**EPC版：** 「EPC项目从开工到竣工」时间轴记录

### 套路二：「神仙网站」——震惊体推荐工具
**原始模式（博主：minhduong7008）：**
标题「一个让XX连夜关门的网站」，30秒展示实用工具

**台球版标题：** 「这个AI台球计分软件，让所有老板连夜更新」
**EPC版标题：** 「这个AI工地管理工具，项目经理看了都沉默」

### 套路三：「科普+干货」——涨知识型内容
**台球版：** 「业余VS职业选手的5个区别」
**EPC版：** 「EPC项目的利润藏在哪3个环节」

### 套路四：「家谱网站」——发掘小众工具推荐
### 套路五：「信息搬运工」——从长视频切爆款涨粉
参考 `references/youtube-clipping-pipeline-full.md`

## 通用执行流程
1. **选题**：按以上套路选择方向
2. **脚本**：AI辅助撰写（标题钩子 + 3个卖点 + 引导收藏/关注）
3. **素材**：录屏/实拍/AI生成
4. **制作**：使用视频产线（脚本→Pillow配图→Edge TTS→FFmpeg合成）
5. **发布**：定时发布到抖音/TikTok
6. **数据复盘**：7天后看播放/收藏/关注数据，迭代优化

## 注意事项
- 标题走「结果展示」或「震惊体」路线
- 前3秒必须有钩子（画面+文案同时）
- 收藏率比点赞率重要
- 前5个视频用来测数据，别追求完美
- 发布节奏：前两周日更，之后隔日更
- 不要搬运抄袭，模式可以学，内容要原创
