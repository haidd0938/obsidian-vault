### 2. 笔 (Bi)

规则：
- 从底分型开始 → 顶分型 → 底分型 → ... 交替
- 顶分型必须高于前一个底分型
- 底分型必须低于前一个顶分型
- 分型之间至少间隔3根K线

### 3. 中枢 (Trading Zone)

定义：至少3笔的重叠区域
- 取第1笔的高点、第2笔的低点、第3笔的高点... 求重叠
- 合并相邻重叠中枢

### 4. 买卖点

| 类型 | 条件 | 置信度 |
|------|------|--------|
| 一类买点 | 价格在中枢下沿附近企稳反弹 | 中 |
| 二类买点 | 突破中枢回踩不破上沿 | 较高 |
| 三类买点 | 远高于中枢，回踩确认 | 高 |
| 一类卖点 | 跌破中枢下沿 | 较高 |

## 艾略特波浪简化实现

通过局部高低点关系推断波浪位置：
- 创新高 → 第3浪或第5浪
- 反弹未创新高 → 第4浪调整
- 创新低 → C浪下跌
- 黄金分割位(0.618)附近 → 回调浪

辅以RSI背离判断（顶背离/底背离）。

## 一目均衡表 (Ichimoku Cloud)

五条线：
1. **转换线(Tenkan-sen)**: (9日高+9日低)/2 — 短期方向
2. **基准线(Kijun-sen)**: (26日高+26日低)/2 — 中期方向
3. **先行线A(Senkou A)**: (转换+基准)/2，前移26日
4. **先行线B(Senkou B)**: (52日高+52日低)/2，前移26日
5. **延迟线(Chikou)**: 当日收盘后移26日

信号判断：
- 云层上方+转换>基准 → 强势多头
- 云层内 → 趋势待确认
- 云层下方+转换<基准 → 弱势空头

## 注意事项

- 缠论需要至少30条K线才能稳定识别分型
- 一目均衡表需要至少60条K线计算52日高低点
- 当前实现是简化版，Vibe-Trading有更完整的：
  - 笔的自适应调整（合并相邻同向笔）
  - 中枢的严格区间套
  - 买卖点的多级别联立
