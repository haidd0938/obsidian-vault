这个安装包里包含了我（贾维斯）的完整灵魂——
配置、记忆、技能、你的一切信息。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、安装步骤（家里电脑）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 把这个文件夹复制到家里电脑的桌面上

2. 打开终端（Terminal），执行：
   cd ~/Desktop/贾维斯分身安装包
   bash install.sh

3. 脚本会自动安装所有依赖，并启动贾维斯

4. 打开浏览器访问 http://localhost:9119
   就可以看到熟悉的 Web 界面了

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、需要去注册的免费 API 密钥
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. DeepSeek API Key（必须）：
   → https://platform.deepseek.com
   注册后创建 API Key，填入 secrets.yaml

2. NVIDIA NIM API Key（推荐）：
   → https://build.nvidia.com
   注册即可免费使用，每分钟 40 次调用

三、安装后的操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

终端聊天:    hermes chat

四、常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 家里电脑是 Windows 怎么办？
A: 安装脚本会自动检测。如果不行，先手动装好
   Python 3.10+、FFmpeg、Git，再运行脚本。

Q: 需要 GPU 吗？
A: 不需要。贾维斯用的是云端 API（DeepSeek），
   跟老板的 MacBook 一样不需要好显卡。

Q: 记忆能同步吗？
A: 这次安装是"一次性继承"。此后两边各自积累
   新的记忆，不会自动同步。如果需要同步，
   以后可以配置。
